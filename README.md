# 一个符号化的笛子数据集

该数据集的收集方法及可视化显示方法可以在另一个仓库下面看到：[A-Method-to-Collect-Jianpu](https://github.com/hrsoup/A-Method-to-Collect-Jianpu) 。通过这个方法，我们收集目前第一个符号化的笛子数据集，包括可以以docx格式显示的简谱以及以MusicXML格式显示的五线谱
